# Art Gallery Data Analysis

# Project Obejctive
  * This project seeks to answer the question: Since everything in today's society is moving towards digital spaces, how has has the Art industry handled this shift?

# Overview
This project analyzes artwork exhibition data to uncover trends in:
- Artwork types exhibited over time
- Highest-rated artworks per year
- Exhibition growth patterns
  
# Key Insights
- Digital art has increased in recent years
- Certain years show peak exhibition activity
- Viewer ratings vary significantly by artwork type

# Visualizations
- Line plots (example image can be viewed in the 'Exhibition_Sizes.png' file)
- Bar charts (example images can be viewed in the 'ArtworkTypes_PerYear.png' and 'Highest_RatedArtwork.png' files)

# Tools Used
- pandas
- matplotlib
- Python

# Dataset
Gallery exhibition dataset containing:
- Titles
- Artists

- Ratings
